# Dev Notes

Repositório pessoal de anotações sobre programação, desenvolvimento de software, bibliotecas, *frameworks*, DevOps e assuntos relacionados à tecnologia.

## Sobre

A ideia deste repositório é manter uma base pessoal de conhecimento, com conteúdos de estudo, referências rápidas, comandos úteis, exemplos de código e observações práticas do dia a dia como desenvolvedor.

## Objetivo pessoal

Este repositório serve como uma base de consulta e aprendizado contínuo. A proposta é registrar conhecimentos de forma simples, organizada e fácil de consultar, criando uma documentação pessoal que possa evoluir com o tempo.
